## Pré-requisitos

- Node.js e npm instalados no seu computador.

## Instalação

### Clone o repositório:

git clone https://github.com/seu-usuario/seu-repositorio.git

### Navegue até o diretório do projeto

### Instale as dependências:
npm install

### Inicie o servidor:

node server.js

### Abra o navegador e acesse:

http://localhost:3000
